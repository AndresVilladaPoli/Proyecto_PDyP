# Proyecto Granja S.A.

## Requisitos
- Python 3.9 o superior
- pip

## Instalación
Crear entorno virtual:

python -m venv venv


Activarlo:

En Windows:

venv\Scripts\activate


Cuando veas algo como (venv) al inicio de la terminal, significa que el entorno está activo.

Instalas dependencias:

pip install -r requirements.txt


Ejecutar la app:

python app.py


Abrir http://127.0.0.1:5000
